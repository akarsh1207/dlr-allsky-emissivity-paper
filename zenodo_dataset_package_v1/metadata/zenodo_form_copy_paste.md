## Description (starter text)
This dataset contains station-wise HDF5 tables for seven NOAA SURFRAD sites (BON, DRA, FPK, GWN, PSU, SXF, TBL), including shortwave and longwave radiative measurements, meteorological variables, quality-control flags, and clear-sky irradiance estimates. The data are structured for all-sky emissivity and downwelling longwave radiation model development and validation. The release includes documentation, schema details, and support scripts for loading and structural validation.
